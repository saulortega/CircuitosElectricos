Proyecto de autoestudio para Circuitos eléctricos I

Acerca de

Este es un proyecto de autoestudio para Circuitos Eléctricos I, Ingeniería de Telecomunicaciones, Politécnico Grancolombiano.

Este grupo está integrado por Saúl Ortega (desarrollo), Kevin Rodríguez (análisis y pruebas), y Rodrigo (logística).

El objetivo de este proyecto es desarrollar una aplicación con capacidad de analizar y desarrollar un circuito electrónico mostrando el paso a paso mediante las leyes de Kirchoff. Inicialmente se prevé que analice y plantee las ecuaciones mediante análisis de nodos. Posteriormente puede ampliarse para realizar también análisis de mallas.

Uso

Arrastre los elementos de la "caja de herramientas" de la parte superior al tablero y dibuje un circuito. Una vez esté hecho el esquema del circuito, pulse el botón "Empezar".

La aplicación validará la correcta estructura del circuito, y, si el circuito se encuentra bien hecho, marcará de manera automática los nodos y las polaridades en las resistencias.

En esta fase de desarrollo la aplicación aún no tiene la capacidad de interpretar valores absolutos dados a los elementos del circuito; sólo dibuja el esquema de nodos y polaridades.

Funcionalidades

Este proyecto se encuentra aún en una fase de desarrollo muy temprana y posee funcionalidades muy limitadas. A continuación se describe las carácterísticas hasta el momento:

  • Valida la correcta o errónea construcción del circuito.
  • Encuentra y marca los nodos no triviales.
  • Crea automáticamente la nomenclatura de los nodos encontrados, según el orden en que se encuentran.
  • Dibuja la polaridad de las resistencias según la nomenclatura de los nodos.
